## Description

Kasirku is a mobile web-based application, and RESTful API designed to support transaction management and cashier operations, as part of the final project of the Web Service Technology course in Semester 5.

## System Requirements

- Linux, Windows or MacOS
- Node JS v18 or later
- npm (Node Package Manager)
- Modern Web Browser (Chrome, Firefox, Edge, Safari)

## Tech Stack

- Next JS
- Tailwind CSS
- Zod Validation
- TypeScript
- Axios
- Tanstack Query

## Security

- Authentication
- Authorization

## Installation

```bash
# Clone project
$ git clone https://github.com/xRiot45/kasirku_frontend_web.git

# Navigate to project repository
$ cd kasirku_frontend_web

# Install all dependencies
$ pnpm install
```

## Setup Environment Variable

```bash
# Change name file
$ cp .env.example .env

# Application
API_URL= # Your API URL
```

## Running The App

```bash
# Watch mode
$ pnpm run start

# Development mode
$ pnpm run dev
```
